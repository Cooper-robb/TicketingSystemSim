# TicketingSystemSim
Simulation of working with a ticketing system
<h2>Summary</h2>

- <b>Creating and assigning tickets by added accounts</b>
- <b>Managing tickets</b>
- <b>Utilizing the dashboard for a centralized overview</b>
- <b>Filtering tickets based on certain criteria</b>

<h2>Environment Used </h2>

- <b>Spiceworks (browser based/cloud-hosted)</b>
